Tappy Tap

Tappy Tap is a crossplatform mobile game, built with Ionic mobile SDK and is using Angular as it's framework.

The main goal of that game is to tap the colored squares as fast as possible and keep going as long as possible in order to reach the highscore. You can also check your global ranking position!

One simple hint is: Follow the black border.

Try out the game for Android

Sunday Football

Sunday Football is a clone of Tappy Tap game but the template is adapter for the soccer fans - like me!

When the white border appears on the balls, you better be fast enough to kick them! In order to get higher at the rankings, you have to keep going as long as possible and as fast as possible.

Good reflexes are the benefit!

Try out the game for Android
